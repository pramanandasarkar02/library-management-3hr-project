# library-management-3hr-project

features/tasks:
1. download one dataset from kaggle and write code to inject data in casendra database
2. open rest api handleling paginations to serve the data
3. write simple one page vite react app to show that data
4. search operation
5. filtering 





stack
- casendra      -> new database
- golang backend 
- typescript frontend (vite)








## database set up 
```bash
docker run -d --name library_cassandra_db -p 9042:9042 cassandra:latest

```
